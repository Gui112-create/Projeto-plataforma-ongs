# Projeto Plataforma ONGs — Entrega I

Este repositório contém as páginas HTML do projeto (index, projetos, cadastro) e os recursos necessários para a Entrega I (HTML5 semântico, formulários e multimídia).

## Como abrir
- Abra `index.html` no navegador (ou use um servidor local).
- O formulário de `cadastro.html` possui validação nativa e máscaras via JavaScript em `js/scripts.js`.

## Estrutura
```
/meu-projeto-ongs
├── index.html          # Página inicial com apresentação da ONG
├── projetos.html       # Lista de projetos e oportunidades
├── cadastro.html       # Formulário de cadastro com validação
├── css/
│   └── styles.css      # Estilos CSS para todas as páginas
├── js/
│   └── scripts.js      # JavaScript com máscaras e validações
└── images/
    └── logo.png        # Logo e outras imagens do projeto
```
